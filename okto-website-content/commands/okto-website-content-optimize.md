Optimize content for PageOptimizer.pro keyword targets using the 5-phase workflow.

## Usage

`/okto-website-content-optimize [content-file]`

If no content file argument is provided, list available content files in content/ and ask the operator which to optimize. The operator must also provide or attach the 4 PageOptimizer brief export files.

## Instructions

@agents/content-manager.md

Enter Optimize mode. Follow the 5-phase workflow: Analyze (parse briefs, count keywords), Plan (propose strategy, WAIT for approval), Rewrite (execute plan), Verify (keyword scorecard), Save+Export (write optimized .md + .html).
