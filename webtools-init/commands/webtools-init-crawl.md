Crawl a URL and return clean markdown content with preserved structure and metadata.

## Usage

`/webtools-init-crawl [URL]`

If no URL argument is provided, ask the operator for the URL to crawl.

## Instructions

@agents/web-crawler.md

Execute the web crawler agent with the provided URL. Pass through any additional context the operator provides as the extraction focus.
